Locked module contract (D4 from CONTEXT.md):
```javascript
// server/game-types/<game>.js
module.exports = {
  id: string,            // type_key, matches games.type_key column
  name: string,          // display name (German)

  // Returns a fresh state for `players` (array of {id, name, emoji, ...gameSpecific}).
  // For Vier Gewinnt, each player includes p.team = 'X' | 'O'.
  // For Anker, optional second arg: config = { maxRunden: number } (default 3).
  initState(players, config?): State,

  // Pure: returns a NEW state, never mutates the input.
  // `value` semantics:
  //   - dreiVollen / plusMinus: raw pin count 0..9 (0 = Pudel)
  //   - grosseHaus / kleineHaus: raw pin count 0..9; meta.slot is 'h'|'z'|'e'
  //   - viergewinnt: column index 0..8; meta.pudel === true for a Pudel turn
  //   - fuchsjagd: raw pin count 0..9
  //   - anker: Anker point value 0,1,2,3,4,5,10
  //   - kda: value = winnerPlayerId of the match identified by playerId (matchId)
  //   - bilderkegel: raw pin count 0..9
  applyThrow(state, playerId, value, meta?): State,

  isFinished(state): boolean,

  // Returns array of result objects. Every object has at least { playerId, score, winner: boolean }.
  // bilderkegel additionally includes { payer: boolean }.
  // viergewinnt result is keyed by team rather than player; see vier-gewinnt.js spec.
  getFinalResults(state): Result[]
};
```
