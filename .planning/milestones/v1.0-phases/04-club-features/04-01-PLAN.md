| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | DELETE /api/game-types/:id | mitigate | WHERE is_builtin = 0 guard in DELETE handler; return 403 otherwise; requireSession enforced |

| T-04-05 | Denial of Service | GET /api/stats computation | accept | Dataset bounded (~10 players, ~200 games/year); synchronous better-sqlite3 computation completes in <100ms |
| T-04-06 | Tampering | POST /api/game-types (duplicate key injection) | mitigate | UNIQUE constraint on game_type_defs.key + catch SQLITE_CONSTRAINT → 409 |
| T-04-SC | Tampering | npm/pip installs | accept | No new packages in Phase 4 — slopcheck not required |
</threat_model>
