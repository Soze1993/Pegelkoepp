game:state payload shapes (set by server/server.js connection handler — updated in Task 2 of this plan):
- Active: `{ idle: false, gameId: number, state: { players: [{ id, name, emoji, wuerfe: [...], ...gameTypeSpecific }, ...], aktSpIdx?: number, ... } }`

- `{ state, finished }`

game:started payload:
- `{ gameId, state, type_key }`
