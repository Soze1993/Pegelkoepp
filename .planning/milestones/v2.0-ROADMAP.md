# Milestone v2.0: Statistiken, Highlights & Turnierbaum

**Status:** ✅ SHIPPED 2026-05-27
**Phases:** 6–9
**Total Plans:** 20
**Production URL:** https://kegel.pegelkoepp.de

## Overview

v2.0 turned Pegelköpp from a game-tracking tool into a full club platform. The core additions: a Double-Elimination tournament engine for "Kegler des Abends", celebratory TV overlays at game end, deep per-player statistics (streaks, head-to-head, historical award counts), a last-evening recap on the homepage, a WhatsApp share button for finished games, and self-hosted fonts so the app works offline at the venue.

Phase 7 expanded significantly beyond its original 4-plan scope to 10 plans — Fuchsjagd, DreiVollen, BK, and VierGewinnt all received dedicated TV renderers with multiple quick-fix iterations, plus audio feedback and game-cancel persistence.

## Milestone Summary

**Key Accomplishments:**

1. Double-Elimination tournament engine (KDA) with live TV bracket tree — 4–12 players, German round labels
2. End-game TV overlays (Kegler des Abends win + BK Verlierer) across all active game types
3. 9 game-type-specific TV renderers with dedicated layouts (FJ, BK, DreiVollen, KDA, VG, PlusMinus, Haus)
4. Year leaderboard + win streaks + H2H + KDA/BK historical counts (7 new stat endpoints)
5. Last-evening recap card on homepage with collapsible game list
6. WhatsApp share link and self-hosted fonts for offline venue operation

**Key Decisions:**

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| TV renderGame: bracket BEFORE players guard | KDA has bracket, not players | ✓ All game types render correctly |
| submitKDAWurfe sequential await | Each throw must complete before next (Pitfall 2) | ✓ No race conditions |
| getBKLoserId in highlights.js | Shared by stats + abende — avoids duplication | ✓ Single source of truth |
| ORDER BY id ASC for game iteration | BK exemption chain requires chronological order | ✓ Correct exemption logic |
| Self-host fonts (6 woff2) | Venue has no internet — must work offline | ✓ No CDN dependency |

**Issues Resolved:**

- FJ: Jaeger ended after 1 round instead of 6 (07-05 fix)
- KDA 4-player bracket: missing L-R1 slot in double-elimination (07-06 fix)
- KDA TV bracket: proportional section sizing + adaptive text (07-09 fix)
