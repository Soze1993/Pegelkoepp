# Requirements Archive: Pegelköpp v2.0

**Archived:** 2026-05-27 — Milestone v2.0 complete
**Original defined:** 2026-05-22
**Core Value:** Ein Tablet tippt, der Fernseher zeigt es sofort — live, ohne Reload.

## v2.0 Requirements (All Complete)

### Statistiken (STATS)

- [x] **STATS-01:** User kann Jahres-Leaderboard aufrufen (Top-Spieler nach Siegen pro Kalenderjahr)
- [x] **STATS-02:** User sieht aktuellen Win-Streak und längsten Win-Streak im Spielerprofil
- [x] **STATS-03:** User kann Head-to-Head-Vergleich zwischen zwei Spielern aufrufen (Siege/Niederlagen direkt gegeneinander)
- [x] **STATS-04:** User sieht im Spielerprofil wie oft er "Kegler des Abends" wurde (historisch, alle Abende)
- [x] **STATS-05:** User sieht im Spielerprofil wie oft er Bilderkegeln verloren hat (historisch, alle Abende)

### Abend-Rückblick (RECAP)

- [x] **RECAP-01:** Startseite zeigt Zusammenfassung des letzten Kegelabends (Datum, Kegler des Abends, Bilderkegeln-Verlierer)
- [x] **RECAP-02:** User kann alle Spiele des letzten Kegelabends durchblättern (Ergebnisse und Gewinner je Spiel)

### Highlights (HIGHLIGHT)

- [x] **HIGHLIGHT-01:** App zeigt Kegler-des-Abends-Symbol in der Spielertabelle nach Spielende
- [x] **HIGHLIGHT-02:** TV zeigt End-Overlay wenn Kegler des Abends feststeht
- [x] **HIGHLIGHT-03:** App zeigt Bilderkegeln-Verlierer-Symbol in der Spielertabelle nach Spielende (wer wenigste Punkte hat)
- [x] **HIGHLIGHT-04:** TV zeigt End-Overlay wenn Bilderkegeln-Verlierer feststeht

### Turnierbaum (TOURNAMENT)

- [x] **TOURNAMENT-01:** User kann "Kegler des Abends"-Spieltyp mit Loser Bracket (Double-Elimination, variabel 4–12 Spieler) starten
- [x] **TOURNAMENT-02:** Bracket-Baum (Winner Bracket + Loser Bracket) wird in der App angezeigt und nach jedem Match aktualisiert
- [x] **TOURNAMENT-03:** Bracket-Fortschritt wird live auf dem TV angezeigt (Socket.io sync)

### TV-Layouts (TV)

- [x] **TV-01:** TV zeigt spieltypspezifisches Layout (Darstellung je nach aktivem Spieltyp angepasst)

### Teilen (SHARE)

- [x] **SHARE-01:** User kann abgeschlossenes Spielergebnis per WhatsApp-Link teilen (generierter Link mit Ergebnis-Zusammenfassung)

### Offline & Performance (OFFLINE)

- [x] **OFFLINE-01:** App lädt Bebas Neue und DM Sans vom eigenen Server (kein Google Fonts CDN erforderlich, funktioniert ohne Internet)
