## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser (untrusted) → /api/stats/year | Query param `year` arrives as user-controlled string |
| Browser (untrusted) → /api/stats/h2h | Query params `a` and `b` arrive as user-controlled strings |
| Browser → /api/stats/streaks, /api/stats/kda-counts, /api/stats/bk-counts | No user-controlled input; data sourced entirely from DB |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-02-S1 | Spoofing | Public read endpoints | accept | No auth required for stats (parity with existing GET /api/stats and /api/highlights/current — TV screen needs them) |
| T-08-02-T1 | Tampering | /api/stats/year SQL injection via year param | mitigate | Validate with `/^\d{4}$/.test(year)` BEFORE binding to prepared statement; reject with 400 on failure. Even if validation were bypassed, parameterized binding (`?`) prevents injection — defense in depth. |
| T-08-02-T2 | Tampering | /api/stats/h2h SQL injection via a/b params | mitigate | Validate via `Number.isInteger(a) && a > 0 && Number.isInteger(b) && b > 0` BEFORE binding; reject with 400. Parameterized binding is also used. |
| T-08-02-I1 | Information Disclosure | Stats expose per-player win history | accept | Same data already exposed by existing GET /api/stats (wins/losses); no new disclosure surface. Internal club app — stats are not sensitive. |
| T-08-02-D1 | Denial of Service | Iteration over all finished games loads every game state into memory | accept | Club app data volume is small (< 5MB DB per CLAUDE.md). Existing /api/stats already does this for every request. Acceptable for current scale. |
| T-08-02-E1 | Elevation of Privilege | Mutation via stats endpoints | mitigate | All endpoints are GET only; no router.post/put/delete added. No DB writes anywhere in stats.js. |
| T-08-02-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed — package legitimacy gate N/A. Imports limited to existing `./highlights` and `./games`. |
